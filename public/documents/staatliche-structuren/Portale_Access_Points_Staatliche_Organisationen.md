## 10. SQL-Beispielabfragen (konzeptionell)

```sql
-- Exterritoriale / souveräne Archive
SELECT * FROM institutions
WHERE (status = 'sovereign_immunity' OR status = 'exterritorial')
  AND (focus = 'restricted_knowledge' OR focus = 'historical_artifacts')
  AND access_level = 'top_secret';

-- Geschlossene Forschungsstädte / Enklaven
SELECT * FROM facilities
WHERE control_body = 'Department_of_Defense'
  AND type = 'isolated_enclave'
  AND funding_source = 'black_budget';

-- SAP / Black Budget / Patent-Geheimhaltung
SELECT * FROM programs
WHERE (legal_basis LIKE '%Executive Order 13526%'
       OR legal_basis LIKE '%DoDD 5205.07%'
       OR legal_basis LIKE '%Invention Secrecy Act%')
  AND classification = 'SAP';
```
